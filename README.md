# halokurs

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Docker
```
docker build -t halo-kurs/latest .
docker run -it -p 8080:8080 --rm --name halo-kurs halo-kurs/latest
```
